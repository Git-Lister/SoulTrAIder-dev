## 🧭 `PROJECT_PLAN.md` – High-Level Roadmap

```markdown
# Soul-TrAIder Project Plan

## Vision

Build an automated system that tracks my geopolitical investment theses, validates predictions, and scales to accommodate new strategies over time.

## Guiding Principles

1. **Data first** – All decisions based on stored, verifiable data
2. **Extensible** – New theses should be addable without core changes
3. **Automated** – Minimize manual updates; let the system work
4. **Insightful** – Provide clear feedback on thesis performance
5. **Learnable** – Use this as a platform to improve my investing through data

## Success Criteria

- [ ] Daily portfolio valuation without manual effort
- [ ] Automated alerts when thesis triggers occur
- [ ] Prediction accuracy tracked and reported
- [ ] News sentiment integrated into thesis assessment
- [ ] Ability to add new theses in < 15 minutes

## Timeline

### Q2 2026 (Mar-May)

- Phase 0-2 complete: automated prices, predictions, alerts
- First 21 predictions tracked to completion

### Q3 2026 (Jun-Aug)

- Phase 3 complete: news + LLM integration
- Begin tracking correlation between news sentiment and price moves

### Q4 2026 (Sep-Dec)

- Phase 4 complete: enhanced dashboard, multi-thesis support
- Expand to cover 3-5 concurrent theses

## Risk Management

- **Data source failure**: Have fallback price sources (Alpha Vantage API key ready)
- **LLM cost**: Use local Ollama to keep costs at zero
- **Time constraints**: Stick to phased approach; each phase delivers value
```
